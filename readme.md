# DSS740 Team 3 — Dropout Prediction Project

## 📌 Project Overview
This project focuses on predicting student dropout using machine learning models. The analysis includes data preprocessing, feature engineering, model training, and evaluation to identify key factors influencing dropout.

---

## 📊 Files in Repository

- **G3_Dropout_Notebook_FINAL.ipynb**  
  Contains full analysis including:
  - Data cleaning
  - Feature engineering
  - Model building (Logistic Regression, Random Forest, etc.)
  - Model evaluation and comparison

- **G3_dropout_predictor_FINAL.py**  
  Python pipeline script that includes:
  - Data preprocessing
  - Outlier detection
  - Model training
  - Prediction workflow

---

## ⚙️ Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

---

## 🎯 Objective
To build an efficient predictive model that can identify students at risk of dropping out and support data-driven decision-making.
